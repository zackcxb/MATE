# MATE Reboot Baseline

This repository is the reboot baseline for inference-first, non-invasive MAS trajectory capture.

## Scope

1. Focus on AgentPipe/AgentLoop orchestration and trajectory capture only.
2. Keep MAS decoupled and non-invasive.
3. Treat training-side integration as out-of-scope for this baseline.

## Kept Assets

1. `AGENTS.md` workflow governance.
2. `skills/` reusable team practices.
3. `docs/plans/2026-03-02-marl-grpo-design.md` cleaned seed design.

## Workflow

1. Start from brainstorming and lock a decision-complete inference-side spec.
2. Implement only after boundaries and interfaces are frozen.
