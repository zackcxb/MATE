# Skill 规范目录

本目录只存放“可执行技能规范”。

## 规范边界

一个 skill 只应包含：
1. 触发条件
2. 输入
3. 执行流程
4. 输出要求或 schema 引用
5. 完成标准

不应包含：
1. 项目进展（放 `docs/project-context.md`）
2. 外部资料清单（放 `docs/references/`）
3. 全局治理规则全文（引用 `AGENTS.md` 即可）
